CH 13. 예외 처리와 C언어와의 링크 지정
1. 오류
- 컴파일 오류(vs 런타임 오류) : 문법에 맞지 않는 문구로 인한 오류
- 실행 오류 : 개발자의 논리 오류, 예외나 대처가 없을 때 발생하는 오류, 실행 결과의 오류

2. 예외와 예외 처리
- 예외란?: 실행 중 프로그램 오동작이나 결과에 영향을 미치는 예상치 못한 상황 발생
- 예외 처리기: 예외 발생을 탐지, 예외를 처리하는 코드(잘못된 결과, 비정상적 실행, 강제 종료 막음)

3. C++ 예외 처리 기본 형식
- try / throw / catch
(1) try { } 블록 : 예외가 발생할 가능성이 있는 코드를 묶음
(2) throw 문 : 발견된 예외 처리를 위해 예외 발생을 알리는 문장. try { } 블록 내에서 이루어짐.
(3) catch { } 블록 : throw에 의해 발생한 예외를 처리하는 코드
